Find Your Career For Life

Team Members: Brandon Bustos, Delaney McDonnell, and Joe Meskin

Last Updated on Sprint 5 on 11/13/2020

Project Proposal

We plan to utilize the information that we are obtaining during Sprint 6 making Matchmaker on a website that will allow you to find the best career path for you between Aviation, Nursing, and Computer Science.

Priority Features

  • Website accessible through a domain name
  • A comprehensive list of questions to determine a compatible career path
  • Have the Matchmaker program run on the website

Secondary Features

  • A complex weighting system that allows for multiple desired answers
  • Have a functioning validate function
  • Record the data from users
  • Display the top results on the site

Languages and Platforms Utilized

  • Python3
  • HTML
  • GoDaddy
  • CSS
  • JavaScript

Technical Features Being Utilized

We did not realize that combining the Matchmaker application and a website was the focus of Sprint 6. However, we stil plan to create this project as we will be learning in class. The additional features are creating a Domain name for our website as well as keeping track of the data from users. We also hope to display a list of the top user results.
