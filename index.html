Find Your Career For Life

Team Members: Joe Meskin and Brandon Bustos

Last Updated on Sprint 5 on 11/7/2020

Project Proposal

We plan to utilize the information that we obtained during Sprint 4 making Matchmaker to create a website that will allow you to find the best career path for you between Aviation, Nursing, and Computer Science.

Priority Features

  • Website accessible through a domain name
  • A comprehensive list of questions to determine a compatible career path
  • Have the Matchmaker program run on the website

Secondary Features

  • A complex weighting system that allows for multiple desired answers
  • Have a functioning validate function

Languages and Platforms Utilized

  • Python3
  • HTML
  • GoDaddy
  • CSS
